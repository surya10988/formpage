4.I have use CSS. in the webpage and there are 3 waysto include css in a webpage as follows

  1. a) Inline CSS: CSS is applied directly to the HTML element using the style attribute. For example: <p style="color: red;">This is a paragraph.</p>
  2. b) Internal CSS: CSS is written inside a <style> tag within the <head> section of the HTML document.
  3. c) External CSS: CSS is written in a separate file with a .css extension, and the file is linked to the HTML document using a <link> tag. Example: <link rel="stylesheet" href="styles.css">.

The reasons css used in a webpage are

  1. a) Separation of Content and Design: CSS allows you to keep the HTML structure clean by separating content (HTML) from design (CSS).
  2. b) Consistent Styling: By using external or internal CSS, you can ensure consistent styling across multiple pages of a website.
  3. c) Responsive Design: CSS enables webpages to adapt to different screen sizes and devices, improving user experience.

  3. c) Cross-Browser Compatibility: JavaScript runs in all modern browsers, making it a universal choice for enhancing webpage functionality.
  4. d) Integration with Other Technologies: JavaScript works seamlessly with HTML and CSS, enabling developers to create interactive and visually appealing web applications.
